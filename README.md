## ❯ Introduction

The Serendipity Identity Service uses Keycloak. Keycloak provides support for OpenID Connect, SAMl 2.0 and OAuth 2.0.
Serendipity's user experience is quite different from Keycloaks so we need to provide a custom Keycloak theme.
